# akagi-bot
Bot for FoxCoalition Discord server

# Commands

Display help:
```
a!help
```

Change your username's color (requires Commodore role):
```
a!color grey
a!color purple
a!color yellow
a!color red
a!color cyan
a!color blue
a!color green
```

Mute someone for 5 minutes (requires administrator role):
```
a!mute 5
```

Add/remove jari role (requires CL, CV or BB role to add):
```
.iam jari squad
.iamn jari squad
```

Display IP address where the bot is running (requires administrator role):
```
a!host
```

Get and reupload images from N recent embed messages or a specific embed message:
```
a!get 5
a!get <Message Link>
```
